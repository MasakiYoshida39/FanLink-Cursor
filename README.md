# FanLink - プロフェッショナル名刺作成アプリ

FanLinkは、美しいデザインの名刺を簡単に作成できるWebアプリケーションです。QRコード機能付きで、モダンなUIとレスポンシブデザインを採用しています。

## 🎯 主な機能

### 📝 名刺作成機能
- **基本情報入力**: 氏名、会社名、役職、連絡先情報
- **住所管理**: 郵便番号付き住所の入力と表示
- **Webサイト**: URL入力と表示
- **QRコード生成**: URLから自動生成されるQRコード

### 🎨 デザインカスタマイズ
- **4つのテンプレート**: クラシック、モダン、ミニマル、クリエイティブ
- **5つのカラーテーマ**: ブルー、グリーン、レッド、パープル、オレンジ
- **リアルタイムプレビュー**: 入力内容が即座に反映

### 📱 QRコード機能
- **QRGenライブラリ**: 高品質で軽量なQRコード生成
- **フォールバック機能**: QRCode.js、シンプルSVGの順でフォールバック
- **高コントラスト**: 純白背景に純黒パターンで読み取りやすさを向上
- **適切なサイズ**: 60px × 60pxで名刺に最適化

### 💾 出力機能
- **HTMLダウンロード**: 完全な名刺HTMLファイルの生成
- **Zoom背景用画像ダウンロード**: 1920×1080ピクセルの高解像度画像をワンクリックで生成
- **名刺ダウンロード(1254×758)ダウンロード**: 1254×758ピクセルの画像をワンクリックで生成
- **印刷対応**: 印刷用CSSスタイル付き
- **レスポンシブ**: 様々なデバイスで表示可能

## 🚀 セットアップと実行

### 前提条件
- モダンブラウザ（Chrome、Firefox、Safari、Edge）
- ローカルWebサーバー（推奨）

### 実行方法

1. **プロジェクトのクローン**
   ```bash
   git clone [repository-url]
   cd FanLink
   ```

2. **ローカルサーバーの起動**
   ```bash
   # Python 3の場合
   python -m http.server 8000
   
   # Node.jsの場合
   npx http-server
   
   # PHPの場合
   php -S localhost:8000
   ```

3. **ブラウザでアクセス**
   ```
   http://localhost:8000/src/main/webapp/
   ```

## 📖 使用方法

### 1. 基本情報の入力
- **氏名**: 必須項目（名刺のメイン情報）
- **会社名**: 所属組織の名称
- **役職**: 職位や役職
- **電話番号**: 連絡先電話番号
- **メールアドレス**: 連絡先メール
- **住所**: 郵便番号付き住所
- **Webサイト**: 会社や個人のWebサイト
- **写真/ロゴ画像**: 顔写真やロゴ画像をアップロードすると、名刺プレビューに即時反映されます（自動リサイズ対応）

### 2. QRコードの設定
- **QRコード用URL**: QRコードに表示したいURLを入力
- 自動的にQRコードが生成され、名刺に表示されます

### 3. デザインのカスタマイズ
- **テンプレート選択**: 4つのデザインテンプレートから選択
- **カラーテーマ**: 5つのカラーテーマから選択
- **リアルタイムプレビュー**: 変更が即座に反映されます

### 4. 名刺の出力
- **名刺HTMLダウンロード**: 「名刺HTMLダウンロード」ボタンでHTMLファイルを保存
- **Zoom背景用画像(1920×1080)ダウンロード**: Zoomのバーチャル背景などに最適な1920×1080ピクセルの画像を保存
- **名刺ダウンロード(1254×758)ダウンロード**: 1254×758ピクセルの画像を保存（バーチャル背景や名刺画像用途）
- **印刷**: ブラウザの印刷機能で印刷可能

## 🐛 トラブルシューティング

### QRコードが表示されない場合
1. **ブラウザのキャッシュをクリア**
2. **「Zoom背景用画像(1920×1080)ダウンロード」ボタンでテスト**
3. **開発者ツールでエラーを確認**

### レイアウトが崩れる場合
1. **ブラウザを最新版に更新**
2. **ハードリフレッシュ（Ctrl+F5）を実行**
3. **異なるブラウザでテスト**

### QRコードが読み取れない場合
1. **URLの形式を確認（https://から始まる）**
2. **QRコードのサイズが適切か確認**
3. **背景とのコントラストを確認**

## 🖼️ 画像機能について

- アップロードできる画像形式は **JPEG, PNG, GIF** です。
- 画像は自動的に最大**300×300px**以内にリサイズされます。
- アップロードした画像は**氏名の右側**に丸型で表示されます。
- 新しい画像を選択すると、前の画像は上書きされます。
- 画像を未選択にすると、名刺プレビューから非表示になります。
- ダウンロード（HTML/PNG）時も、プレビューに表示されている画像がそのまま出力されます。

## 🔒 セキュリティ・プライバシー

- FanLinkは**完全クライアントサイド**で動作し、入力した個人情報や画像データは**外部サーバーに送信されません**。
- 画像や名刺データは、ユーザーのブラウザ内でのみ処理・保存されます。
- プライバシーを重視した設計のため、安心してご利用いただけます。
